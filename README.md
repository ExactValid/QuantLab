# Quant_Lab (量化交易与策略研究实验室)

基于机器学习与数学规划的量化投资策略回测系统。项目实现了从数据接入、模型训练、仓位优化到前端看板可视化的完整量化研究管线。

## 项目介绍

本项目主要包含以下核心模块与技术栈：

1. **数据接入**：利用 AkShare 等数据接口，自动化获取并清洗金融历史 K 线与因子数据。
2. **机器学习策略**：构建基于随机森林（Random Forest）与梯度提升（Gradient Boosting/LightGBM）等机器学习算法的样本学习模型，对市场趋势或收益率进行预测。
3. **仓位优化**：引入 KellySizer（凯利公式仓位控制）与 Sigmoid 映射函数，并结合 Gurobi 数学规划求解器，对预测结果进行多资产组合的仓位分配与风险对冲优化。
4. **回测与可视化**：运行底层回测引擎生成交易表现数据，并通过 JavaScript 语言及相关图表库渲染出交互式的量化回测看板。
5. **在研功能**：当前已引入贝叶斯优化（Bayesian Optimization）算法，正在针对机器学习模型的超参数进行自动化调优尝试。

## 环境配置

本项目采用了标准 Python 包管理模式，支持通过可编辑模式（Editable Mode）直接一键安装项目及其核心依赖。

### 1. 外部求解器依赖
项目中的高级仓位优化功能依赖 Gurobi 求解器，运行前请确保：
* 已安装 [Gurobi Optimizer](https://www.gurobi.com/) 客户端软件。
* 目前的复杂度暂不需要配置并激活有效的 Gurobi 授权许可（License）。

### 2. 命令行安装步骤
在确保 Python 环境就绪后，打开终端执行以下命令：

```bash
git clone <你的仓库地址>
cd Quant_Lab
pip install -e .
```

## 使用指南

请按照以下流水线顺序运行脚本以执行完整的策略研究生命周期：

### 1. 数据准备
运行数据下载脚本获取 A股数据（注：需提前在代码中手动指定目标股票/ETF 代码）：

```bash
python scripts/download_data.py
```

### 2. 模型训练与管理
启动机器学习样本学习流程，运行后系统会弹出交互式界面供选择当前已有的机器学习模型：

```bash
python scripts/train/train_model.py
```

* 训练完成后，模型的最优参数将自动保存至 `scripts/train/model_config.py` 中，以便后续追溯与手动微调。
* 如需对本地已生成的模型进行清理，可运行模型管理脚本（支持模型删除等功能）：

```bash
python scripts/train/manage_model.py
```

### 3. 策略回测与画布渲染
启动策略回测引擎：

```bash
python scripts/strategy/run_backtest.py
```

* 运行后将弹出交互式窗口引导选择优化器（包含单股仓位优化与多股仓位优化）。
* 回测引擎运行完毕后，将自动联动 `html_renderer.py`，在 `outputs/reports/` 目录下渲染出交互式的 JS 回测数据看板。
